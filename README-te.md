DOCX ఫైల్ కంటెంట్ ఎక్స్‌ట్రాక్షన్ మరియు డిస్‌ప్లే కోసం ఆధునిక వెబ్ అప్లికేషన్. Next.js, TypeScript, మరియు Tailwind CSS తో నిర్మించబడింది, బహుభాషా మద్దతు మరియు SEO ఆప్టిమైజేషన్‌తో.

## 📋 సంక్షిప్త వివరణ

DOCX Unpacker Web App అనేది వెబ్ అప్లికేషన్ ఇది వినియోగదారులను Microsoft Word డాక్యుమెంట్లను (.docx) అప్‌లోడ్ చేయడానికి మరియు వెబ్ బ్రౌజర్‌లో వాటి కంటెంట్‌ను నేరుగా చూడడానికి అనుమతిస్తుంది. ఈ అప్లికేషన్ డాక్యుమెంట్ కంటెంట్‌ను ఎక్స్‌ట్రాక్ట్ చేసి, దానిని నిర్మాణాత్మక, చదవగలిగే ఫార్మాట్‌లో ప్రదర్శిస్తుంది.

### ✨ ముఖ్య లక్షణాలు

- 📄 **DOCX ఫైల్ అప్‌లోడ్**: డ్రాగ్ అండ్ డ్రాప్ లేదా క్లిక్ చేసి ఫైల్ ఎంపిక
- 🔍 **కంటెంట్ ఎక్స్‌ట్రాక్షన్**: డాక్యుమెంట్ టెక్స్ట్, చిత్రాలు మరియు టేబుల్స్ ఎక్స్‌ట్రాక్షన్
- 🎨 **రెస్పాన్సివ్ డిజైన్**: అన్ని పరికరాలకు ఆప్టిమైజ్డ్ డిస్‌ప్లే
- 🌍 **బహుభాషా మద్దతు**: 21 భాషల మద్దతు
- ⚡ **వేగవంతమైన ప్రాసెసింగ్**: క్లయింట్-సైడ్ సమర్థవంతమైన ఫైల్ ప్రాసెసింగ్
- 🔒 **గోప్యత-కేంద్రీకృత**: ఫైల్స్ స్థానికంగా ప్రాసెస్ చేయబడతాయి, సర్వర్‌కు పంపబడవు
- 🎯 **SEO ఆప్టిమైజ్డ్**: సెర్చ్ ఇంజిన్‌ల కోసం ఆప్టిమైజ్డ్
- 📱 **PWA రెడీ**: ప్రోగ్రెసివ్ వెబ్ అప్లికేషన్ లక్షణాలు

## 🛠️ టెక్నికల్ స్టాక్

- **ఫ్రేమ్‌వర్క్**: Next.js 14 (App Router)
- **భాష**: TypeScript
- **స్టైలింగ్**: Tailwind CSS
- **UI కాంపోనెంట్స్**: Radix UI
- **ఐకాన్స్**: Lucide React
- **ఫైల్ ప్రాసెసింగ్**: JSZip, xml2js
- **అంతర్జాతీయీకరణ**: next-intl
- **డిప్లాయ్‌మెంట్**: Vercel
- **AI ఇంటిగ్రేషన్**: Google Genkit (ఐచ్ఛిక)

## 🚀 ప్రారంభించడం

### ముందస్తు అవసరాలు

- Node.js 18.0 లేదా అంతకంటే ఎక్కువ
- npm, yarn, pnpm, లేదా bun

### ఇన్‌స్టాలేషన్

1. రిపోజిటరీని క్లోన్ చేయండి:
```bash
git clone https://github.com/yourusername/docx-unpacker-webapp.git
cd docx-unpacker-webapp
```

2. డిపెండెన్సీలను ఇన్‌స్టాల్ చేయండి:
```bash
npm install
# లేదా
yarn install
# లేదా
pnpm install
```

3. డెవలప్‌మెంట్ సర్వర్‌ను ప్రారంభించండి:
```bash
npm run dev
# లేదా
yarn dev
# లేదా
pnpm dev
```

4. బ్రౌజర్‌లో [http://localhost:3000](http://localhost:3000) తెరవండి
