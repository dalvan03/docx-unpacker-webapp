- Next.js 15, TypeScript, Tailwind CSS, shadcn/ui
- உலாவியில் JSZip மூலம் `.docx` திறந்து பார்வையிடலாம்
- கோப்பு மரம், பட முன்னோக்கு, raw XML, எளிய உரை முன்னோக்கு
