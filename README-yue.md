# Docx Unpacker Webapp — Cantonese (yue)

主要文件：`README.md`（英文）。葡萄牙語（巴西）：`README-pt-br.md`。

呢個檔係用嚟做粵語版 README。非常歡迎幫手翻譯同維護。

簡介
- Next.js 15、TypeScript、Tailwind CSS、shadcn/ui
- 用 JSZip 喺瀏覽器解包 `.docx`
- 檔案樹、圖片預覽、原始 XML、簡單文字預覽
- 多語言同 locale-based SEO
